Project

Berikut adalah beberapa project yan telah saya buat dan saya akan terus belajar lagi agar saya dapat mengembangkan skils saya pada bidang ini.

Website Kenangan Senja

Web Kenangan Senja dirancang menggunakan HTML, CSS, dan JavaScript. HTML digunakan untuk struktur dasar, CSS memberikan desain minimalis dan responsif, sementara JavaScript menambahkan fungsionalitas interaktif seperti berbagi kenangan senja dan animasi halus. Dengan teknologi ini, web ini menghadirkan pengalaman yang indah dan berpartisipasi dalam merayakan kecantikan matahari terbenam.

Website Sumatra Barat

Website Sumatra Barat menonjol dengan desain modern menggunakan HTML, CSS, dan JavaScript. HTML memberikan struktur yang kuat, CSS menyajikan tampilan yang bersih dan menawan, sedangkan JavaScript menambahkan interaktivitas. Situs ini merangkum keindahan dan warisan Sumatra Barat melalui kombinasi teknologi yang mengundang pengunjung untuk terlibat.

Website Perpustakaan

Website Perpustakaan dikembangkan menggunakan Bootstrap untuk tampilan responsif dan CodeIgniter sebagai kerangka kerja backend. Dengan Bootstrap, situs ini memiliki desain yang responsif dan mudah diakses di berbagai perangkat. CodeIgniter memberikan struktur backend yang efisien, mengelola data perpustakaan dan memungkinkan pencarian yang efektif. Platform ini menghadirkan pengalaman online yang nyaman untuk menelusuri dan mencari buku di perpustakaan.
